# 🎮 Exercício - Jogo de Perguntas

## 📌 Descrição
Este projeto consiste no desenvolvimento de um jogo de perguntas e respostas, onde o jogador escolhe um tema e responde a um conjunto de questões.

---

## 🧠 Funcionalidades

### 🎯 Seleção de Tema
Ao iniciar o jogo, o jogador deve selecionar um tema para as perguntas.

**Exemplos de temas:**
- História, Física, Geografia e Matemática  
- Java, Kotlin, Python e Swift  
- Livros, Filmes, Séries e Documentários  

---

### ❓ Perguntas e Respostas
- Após a escolha do tema, serão feitas **5 perguntas** relacionadas a ele.
- Cada pergunta possui **4 opções de resposta**, sendo **apenas uma correta**.
- Após responder:
  - ✅ Informe se a resposta está correta  
  - ❌ Caso esteja incorreta, exiba a resposta correta  

---

### 🏆 Pontuação e Repetição
- Ao final das 5 perguntas:
  - Mostre a quantidade de respostas **corretas** e **incorretas**
- Em seguida:
  - Exiba novamente os temas para reiniciar o jogo  

---

## 🚀 Sugestões de Melhorias (Opcional)

### 🔀 Perguntas Aleatórias
- Gere perguntas em ordem aleatória a cada execução  

### 📚 Maior Banco de Perguntas
- Tenha mais de 5 perguntas por tema  
- Permita maior variedade e repetição controlada  

### 🏅 Ranking
- Exiba o nome do jogador e sua pontuação final  
- Utilize um **Dialog** para mostrar o resultado  

### 🎚️ Níveis de Dificuldade
- Implemente níveis como:
  - Fácil
  - Médio
  - Difícil  
- Ajustes possíveis:
  - Número de opções
  - Tempo para responder
  - Complexidade das perguntas  

### ⏭️ Opção de Pulo
- Permita que o jogador pule perguntas  
- Defina um limite de pulos por jogador  

---

## 🛠️ Objetivo
Praticar conceitos de lógica de programação, estruturas condicionais, controle de fluxo e interação com o usuário.
